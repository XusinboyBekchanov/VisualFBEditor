Language file structure:
English - Name of language in native language
[Compiler] - Texts reported by the compiler
Text = Translated text
[General] - IDE interface texts
Text = Translated text
[Keywords] - FreeBasic Keywords
Text = Translated text
[Property] - Texts in the Properties Window
Name = Translated name|Translated description
[Templates] - Templates
Text = Translated text

This file must be saved in UTF8 (BOM) encoding to be read correctly by other language systems.

The "&" symbol can be placed anywhere in the translated text, taking into account the peculiarities of the language.

For tests you can use tester.lng file.

Some languages are machine translated and can be improved.

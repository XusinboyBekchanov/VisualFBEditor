MediaPlayer

It's a Directshow Media Player, translate from dx8 directshow\players\playwnd

Funtion added 2024/02/22:
1. Screenshot
2. Fast Forward/Slow Forward
3. Play to desktop monitr 1 or more
4. Scale 1/4, 1/2, 1/1 and fullscreen

## Definition
Namespace: [`My.Sys.Forms`](My.Sys.Forms.md)

`ComboBoxEx` - `ComboBoxEx` is a Control within the MyFbFramework, part of the freeBasic framework. <br> `ComboBoxEx` - ComboBoxEx controls are combo box controls that provide native support for item images.

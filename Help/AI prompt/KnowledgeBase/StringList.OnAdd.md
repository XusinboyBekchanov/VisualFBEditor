# StringList.OnAdd Event

`OnAdd` is event of the StringList control, part of the freeBasic framework MyFbFramework.
## Syntax
```freeBasic
OnAdd As Sub(ByRef Sender As StringList, FItem As String, FObj As Any Ptr = 0)
```
